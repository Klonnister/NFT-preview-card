### What I learned

I practiced centering the card in the webpage

```css
body{
    min-height: 100vh;
    width: 100%;
    display: flex;
    justify-content: center;
    align-items: center;
}
```

Learned how to use the property inset in css to center the image:

```css
.view-img {
    position: absolute;
    inset: 0;
    margin: auto;
}
```
